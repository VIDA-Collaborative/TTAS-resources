Teacher Time Analysis & Supports — Phase 1

Data Collection & Analysis Processes

This document outlines the step-by-step data collection, cleaning, synthesis, and analysis process for each data collection tool used during Phase 1, along with roles, resources, and how the data is used.

Key definitions
Data collection
The process of gathering and measuring information from various sources.
Data cleaning
The process of identifying and correcting or removing inaccurate, incomplete, or duplicate data from a dataset.
Data synthesis
The process of combining information from multiple sources.
Data analysis
The practice of systematically inspecting raw data to uncover patterns, draw conclusions, and inform decision-making.
