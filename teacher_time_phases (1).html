Teacher Time Analysis & Supports

Project Phases: Assess, Plan, and Implement

A structured, three-phase approach to analyzing how teachers allocate their time—identifying inefficiencies and developing actionable strategies for workload optimization.

Phase 1
Assess Needs
Identify where the district is currently at

The district quality team and teacher-researchers collect quantitative and qualitative data to assess district needs. Findings establish the evidence base for all subsequent phases and for performance management.

Teacher time diaries Teacher focus groups Teacher & admin survey Baseline monitoring data
Strategies — Training & Coaching Topics
  • Collecting and analyzing focus group data, time audit data, survey, and baseline data
  • Data synthesis, analysis, and sense-making
  • Preparing for the June convening
  • Having tough data conversations
  • Creating a needs assessment
  • Teacher-researchers' evolving role in the project
Resources & Tools
  • Focus group data collection guide
  • Time audit data collection guide
  • Survey data collection guide
  • Baseline data collection template
  • Campus synthesis templates
  • District data analysis protocols
  • Needs assessment template
Milestones
  • Teacher-researchers synthesize findings into a joint presentation of findings
  • Districts submit a needs assessment using information collected from the case study campuses
Output: Needs Assessment
Needs assessment informs ideal state planning
Phase 2
Identify Ideal State & Strategies
Identify where you want the district to be and how it can get there

Using the Phase 1 needs assessment and a resource library, the district quality team collaborates to define an ideal state and the strategies needed to reach it.

Strategies — Training & Coaching Topics
  • Conducting a root cause analysis
  • Identifying your ideal future state
  • Identifying strategies to close the gap between current and ideal state
  • Developing your theory of action
Resources & Tools
  • TTAS resource library
  • Theory of action template
Milestones
  • Districts submit a theory of action identifying an ideal state and strategies to move from current state toward the ideal state
Output: Theory of Action
Theory of action drives implementation planning
Phase 3
Create an Implementation Plan
Create a detailed plan for how the district will reach its ideal state

Using the theory of action developed in Phase 2, the district quality team works together to create a detailed, actionable implementation plan for optimizing teacher time and workload management.

Strategies — Training & Coaching Topics
  • Creating an implementation plan
  • Developing a measurement plan
  • Establishing performance management routines
  • Preparing for project launch
Resources & Tools
  • Implementation plan template
Milestones
  • Districts submit a detailed implementation plan in response to the findings of the needs assessment and theory of action
Output: Implementation Plan
